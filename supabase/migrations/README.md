## đź› ď¸Ź Senior Maintenance Standards

> [!IMPORTANT]
> **Idempotency is Mandatory**
> Every script must be safe to run multiple times. Always use `CREATE TABLE IF NOT EXISTS`, `CREATE OR REPLACE`, and `ALTER TABLE ... ADD COLUMN IF NOT EXISTS`.

> [!TIP]
> **Scaling to 1,000 DAU**
> Avoid `SELECT COUNT(*)` on huge tables in your application code. Always prefer the **Trigger-to-Table** aggregation pattern (see Migration #13) to keep the UI instant.
